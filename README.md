# StudentsMart

**StudentsMart** is a PHP-based web application built for managing student-related products and services.

## Features
- User registration and login
- Product listing and management
- Simple shopping cart
- Admin panel for managing products

## Tech Stack
- PHP
- MySQL (for database)
- HTML, CSS, JavaScript
- XAMPP (for local development)
